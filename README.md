# Perfume Store

A modern, interactive perfume store website with multi-language support, theme switching, user authentication, shopping cart functionality, and an automatic sliding top sold products carousel.

## Features

- ✅ **Modern UX/UI**: Responsive design with smooth animations, gradients, and Font Awesome icons
- ✅ **Interactive Functions**: Product search, category filtering, shopping cart with add/remove functionality
- ✅ **Top 10 Products Slider**: Automatic sliding carousel showcasing the top 10 best-selling products
- ✅ **User Authentication**: Login and registration system with local storage
- ✅ **Theme Settings**: Light/dark mode toggle with smooth transitions
- ✅ **Order History**: Complete order tracking for logged-in users
- ✅ **Multi-language Support**: English, Khmer, and Chinese translations
- ✅ **Responsive Design**: Works perfectly on desktop and mobile devices

## Technologies Used

- HTML5 with semantic structure
- CSS3 (with CSS Grid, Flexbox, animations, and gradients)
- JavaScript (ES6+ with modern DOM manipulation)
- Font Awesome for icons
- Local Storage for data persistence

## How to Run

1. Open `index.html` in a modern web browser
2. The website will load with 36 sample products and start the auto-sliding carousel

## Usage

### Browsing Products
- View featured products on the main page (36 total products available)
- Use the search bar to find specific perfumes
- Filter products by category (Floral, Woody, Oriental, Fresh)
- Check out the top 10 products slider with automatic transitions

### Shopping
- Click "Add to Cart" on any product (from main grid or slider)
- View cart by clicking the cart icon in the header
- Remove items from cart or proceed to checkout

### User Account
- Click "Login" to sign in or create a new account
- Registered users can view their order history
- User data is stored locally in the browser

### Settings
- Toggle between light and dark themes with the moon/sun icon
- Switch languages using the dropdown menu
- Preferences are saved automatically

### Top 10 Products Slider
- Automatic sliding carousel every 4 seconds showing top 10 best-sellers
- Manual navigation with arrow buttons
- Hover to pause auto-sliding
- Shows product ratings and sales count
